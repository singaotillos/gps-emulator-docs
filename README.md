##  Technical Specifications

### Architecture
- **Backend**: Python 3.8+ (Flask)
- **Frontend**: HTML5, Bootstrap 5, jQuery
- **Database**: SQLite (PostgreSQL compatible)
- **API**: RESTful + WebSocket
- **Protocols**: TCP/UDP/HTTP

- Handles 100+ concurrent devices
