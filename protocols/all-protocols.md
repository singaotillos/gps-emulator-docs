## üöÄ Adding Custom Protocols

Want to add your own protocol? The emulator is extensible!

### Steps to Add Protocol

1. Create protocol directory: `protocols/yourprotocol/`
2. Implement `simulator.py` (message generation)
3. Implement `client.py` (network communication)
4. Add to protocol registry
5. Test with your platform

See `protocols/tk103/` for a complete example.
